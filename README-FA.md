# افزونه بومی OpenWrt (LuCI) برای AdGuard Home

🌍 [English Version](README.md)

یک پلاگین بسیار سریع و بومی برای OpenWrt LuCI که دامنه‌های CDN و شبکه‌های ابری را به صورت خودکار ترجمه کرده و IPهای محلی آن‌ها را مستقیماً به بخش **DNS Rewrites** در AdGuard Home تزریق می‌کند.

## 💡 ایده پروژه
سرویس AdGuard Home یک مسدودکننده و مدیریت‌کننده قدرتمند DNS است، اما هدایت دامنه‌های CDN (برای دور زدن محدودیت‌ها یا بهینه‌سازی مسیرها در Passwall/SmartDNS) معمولاً نیازمند اسکریپت‌های سنگین پایتون یا تنظیمات دستی طاقت‌فرساست.

این افزونه با استفاده از **اسکریپت‌های بومی شل (`sh`) و سیستم قدرتمند UCI در OpenWrt** این مشکل را حل کرده است. این سیستم دامنه‌های مشخص شده را می‌خواند، آن‌ها را به صورت همزمان (موازی) با استفاده از سیستم روتینگ داخلی روتر ترجمه می‌کند و بلاک IPهای به دست آمده را در کمتر از یک ثانیه به فایل `AdGuardHome.yaml` تزریق می‌کند؛ بدون نیاز به نصب پایتون یا پکیج‌های سنگین!

## ✨ ویژگی‌ها
- **بدون پیش‌نیازهای سنگین**: نوشته شده با Shell و Lua. بدون نیاز به پایتون.
- **سرعت فوق‌العاده**: استفاده از قابلیت پردازش پس‌زمینه موازی (`&` در Bash) برای ترجمه همزمان صدها دامنه.
- **رابط کاربری یکپارچه**: ادغام کامل با پنل تنظیمات LuCI (`Services -> ADG DNS Lookup`).
- **استاندارد UCI**: مدیریت لیست دامنه‌ها با استفاده از سیستم استاندارد UCI به جای فایل‌های متنی نامنظم.
- **زمان‌بندی داخلی (Cron)**: قابلیت تنظیم آپدیت‌های خودکار (مثلاً هر ۳ ساعت یا روزانه) برای تازه نگه‌داشتن IPها.
- **لیست‌های آماده و جامع**: دارای دسته‌بندی‌های پیش‌فرض برای دامنه‌های Cloudflare, Fastly, Google, Meta, Microsoft و...
- **تزریق ایمن به YAML**: استفاده از نشانگرهای دقیق متنی (`awk`/`sed`) برای جلوگیری از خرابی تنظیمات AdGuard.

## 🚀 نصب آسان
برای نصب آخرین نسخه کامپایل شده به صورت خودکار، دستور زیر را در ترمینال OpenWrt وارد کنید:
```sh
opkg update
opkg install https://github.com/mortezabahmani/luci-app-adg-dnslookup/releases/download/v1.0.0/luci-app-adg-dnslookup_1.0.0-1_all.ipk
```

## 🛠️ راهنمای گام‌به‌گام بیلد از سورس
اگر ترجیح می‌دهید پکیج را خودتان با استفاده از OpenWrt SDK کامپایل کنید:

1. **دانلود ایمیج داکر OpenWrt SDK** (نسخه معماری روتر خود را جایگزین کنید، مثلاً `x86_64`):
   ```sh
   docker pull openwrt/sdk:x86-64-23.05.0
   ```
2. **اجرای کانتینر SDK**:
   ```sh
   docker run --rm -it openwrt/sdk:x86-64-23.05.0 bash
   ```
3. **کلون کردن ریپازیتوری در پوشه پکیج‌ها**:
   ```sh
   cd package
   git clone https://github.com/USERNAME/luci-app-adg-dnslookup.git
   cd ..
   ```
4. **انتخاب پکیج در منوی تنظیمات**:
   ```sh
   make menuconfig
   # به مسیر LuCI -> 3. Applications -> luci-app-adg-dnslookup بروید و دکمه 'M' را بزنید.
   ```
5. **کامپایل پکیج**:
   ```sh
   make package/luci-app-adg-dnslookup/compile -j$(nproc) V=s
   ```
6. **دریافت فایل نهایی**:
   فایل `.ipk` یا `.apk` در مسیر `bin/packages/*/base/` ایجاد خواهد شد. آن را به روتر منتقل کرده و نصب کنید.

## 📂 ساختار کدها
- `/Makefile`: فایل استاندارد بیلد پکیج OpenWrt.
- `/luasrc/`: شامل کدهای رابط کاربری (Lua CBI Models و Controller).
- `/root/etc/config/adg_dnslookup`: فایل تنظیمات مبتنی بر استاندارد UCI (حاوی لیست دامنه‌ها).
- `/root/etc/init.d/adg_dnslookup`: سرویس زمان‌بندی (Cron) و راه‌اندازی.
- `/root/usr/bin/adg-sync.sh`: هسته اصلی سیستم (Shell Script) برای پردازش موازی و تزریق مقادیر.
